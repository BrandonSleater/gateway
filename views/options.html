    • Kickstarter

    Kickstarter is a popular crowdfunding website which focuses on funding project based ideas

    Pros


    • Kickstarter is a popular site it has over1 billion dollars promised to projects.

    • Kickstarter has and atomic model for funding either the project is fully funded or the project recieves no funding. Which creates a system that avoids half funded projects.

    • Allows for the backing of a wide variety of projects that fit into any of these cagegories: Art, Comics, Dance, Design, Fashion, Film, Food, Games, Music, Photography, Publishing, Technology, and Theater.

    Cons


    • Kickstarter takes 5% of all funding of a funded project.

    • Kickstarter does not allow for charity projects or projects that use charity donations as an incentive.

    • The projects are restricted to these cagegories: Art, Comics, Dance, Design, Fashion, Film, Food, Games, Music, Photography, Publishing, Technology, and Theater.

    • Kickstarter does not allow for the use of equity as a reward/incentive for backing a project.

    • Kickstarter does not allow for multiple rewards per pledge(funding promise).

    Requirements


    • Projects with hardware require physical prototype

    • Kickstarter requires that you show only what you have done and only allows one reward(incentive) per pledge(funding).
